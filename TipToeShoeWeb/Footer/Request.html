Request Return

Shoe Warehouse is confident that you will be happy with your purchase. If for any reason this is not the case, we offer returns within 28 days of the order date for a full refund (less any shipping costs paid).

We have 3 conditions for returns due to a change of mind:

1. Items must be returned within 28 days of the order date;

2. Items must be unworn with the original tags still attached - we are unable to accept returns with dirty marks or wear marks;

3. Items must be purchased directly from our website.
